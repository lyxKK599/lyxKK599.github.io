第十届省赛JAVA b组

试题C 数列求值

public static void main(String[] args) {

        int cnt = 4;

        int a = 1 ;

        int b = 1 ;

        int c = 1 ;

        while(cnt!=20190324) {

            if(cnt%3 == 0) {

                a = a + b + c;

                a%=10000;

            }

            if(cnt%3 == 1) {

                b = a + b + c;

                b%=10000;

            }

            if(cnt%3 == 2) {

                c = a + b + c;

                c%=10000;

            }

            cnt ++;

        }

        System.out.println((a+b+c)%10000);

        System.out.println("a = "+a);

}

这种题用迭代 别傻傻的开数组了

有道题用深搜BFS 以后再搞

后缀表达式 要分很多种情况讨论 感觉大部分题还是有思路的 但是真不一定写得出来 害

转义字符!!! 我测试下

s.split(“\\s+|\\.”);

说实话 看不太懂 但是效果是把.跟左右的空白或者空白分隔开了

s.split(“\\s+|\\2”);

这样就把字符串“2”跟左右的空白分割开了 感觉可以记者这样用吧

如果有多个空格要分隔开的可以用s.split(“\\s+”);正则表达式 之前没接触过
